# React use() + Suspense (Vite + TypeScript)

## Description
Навчальний проєкт на React (Vite + TS), який демонструє отримання даних з Promise через `use()` у компоненті та керування станами завантаження через `Suspense`. Помилки Promise відображаються через Error Boundary.

## Install & Run

```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
